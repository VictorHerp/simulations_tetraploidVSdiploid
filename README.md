# Simulations_tetraploidVSdiploid
This repository contains scripts to simulate autotetraploid and diploid populations using SLiM. The outputs include genetic statistics such as masked load, realized load, and fitness. 

To run the bottleneck and stable population simulations you need to load the files "Burn-in_hs-input_partial_recessive_mut_tetraploids" and "Burn-in_hs-input_partial_recessive_mut_diploids" into defineConstant("hs_input" ...) and defineConstant("hs_input_dip"...) respectively.

Additionally, this repository includes R scripts for plotting the simulation results.
